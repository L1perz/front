# Интернет-магазин (Практики №9–11)

## Описание

Проект представляет собой веб-приложение интернет-магазина с разделением на frontend и backend.

Реализованы:
- Авторизация и регистрация пользователей
- Система ролей (user, seller, admin)
- CRUD операции для товаров
- Админ-панель для управления пользователями
- Блокировка пользователей
- Swagger-документация API

---

## Технологии

### Backend
- Node.js
- Express
- JWT (авторизация)
- bcrypt (хэширование паролей)
- Swagger (документация API)

### Frontend
- React
- Axios
- CSS / SASS

---

## Роли пользователей

- **Гость**
  - Регистрация
  - Авторизация

- **Пользователь (user)**
  - Просмотр товаров

- **Продавец (seller)**
  - Просмотр товаров
  - Добавление товаров
  - Редактирование товаров

- **Администратор (admin)**
  - Все возможности продавца
  - Удаление товаров
  - Управление пользователями
  - Изменение ролей
  - Блокировка пользователей

---

## Функциональность

### Аутентификация
- Регистрация
- Вход
- Получение текущего пользователя (/api/auth/me)
- JWT access и refresh токены

### Товары
- Получение списка товаров
- Получение товара по ID
- Создание товара (seller, admin)
- Редактирование товара (seller, admin)
- Удаление товара (admin)

### Пользователи (только admin)
- Получение списка пользователей
- Получение пользователя по ID
- Обновление данных пользователя
- Изменение роли
- Блокировка/разблокировка
